# Millie

A fast, private macOS browser — the Chromium engine with a native SwiftUI interface (spaces, split view, tab tear-off, and more).

## Download

**[⬇︎ Download the latest Millie for macOS](https://github.com/redclayai/millie/releases/latest)** — Apple Silicon, macOS 12+. Signed & notarized by Red Clay AI, Inc.

Open the `.dmg`, drag **Millie** to Applications.

## Updates

Millie updates itself automatically. You can also check manually in **Settings → About → Check for Updates…**.
